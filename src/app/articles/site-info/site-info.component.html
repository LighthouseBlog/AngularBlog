About

This site was developed by Sam Pastoriza and James Edwards. We wanted a blog that was self sustaining and easily modifiable. We are using the Angular framework developed by Google and Angular Material to style and layout the site. An express based backend application utilizes MongoDb and Redis to store data on users and articles. The blog is completely open-sourced on GitHub and contributers are more than welcome. The end goal is a fully functional blog mainly covering technical topics, but can be reused as a base for other blogs.
